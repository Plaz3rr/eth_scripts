Самодельные скриптецы
Blur_deposit.py - Скрипт отправляет рандомные суммы ETH в заданных пределах, поочередно с рандомной заданной задержкой, на контракт Blur, активируя функцию Deposit. Проверяет газ каждые 10сек и отправляет когда падает до заданных пределов. Для его работы создаем файлы blur_dep.txt (для приватников), last_successful_tx.txt (для записи посл.транзакции - чтобы после остановки скрипта всегда можно было увидеть посл.успешную транзу)
